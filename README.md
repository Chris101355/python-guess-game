# 🐍 我的第一個 Python 小遊戲：猜數字
這是我在資圖系「資料庫概論」課程中，結合 AI 協助完成的第一個程式專案。

## 🎮 遊戲簡介
電腦會隨機產生一個 1 到 100 之間的數字，玩家需要透過提示（太大或太小）來找出正確答案。

## 🛠️ 使用技術
* **語言：** Python 3
* **開發工具：** Visual Studio Code
* **版本控制：** GitHub
